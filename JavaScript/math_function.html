Math Functions in JavaScript

1. Math.round():

  • Rounds a number to the nearest integer.
  • If decimal ≥ 0.5 → rounds UP
  • If decimal < 0.5 → rounds DOWN

2. Math.floor():

  • Always rounds DOWN to the nearest integer.
  • Useful when you always want a lower value (ex: index, pagination).

3. Math.ceil():

  • Always rounds UP to the next nearest integer.
  • Useful when you always want a higher value.

4. Math.random():

  • Generates a random decimal number between 0 (inclusive) and 1 (exclusive).
  • To get a random number within a specific range, you can scale and shift the result.

5. Math.abs():

  • Returns the absolute value of a number (non-negative value).
  • Used when only magnitude is needed, not sign.

6. Math.sqrt():

  • Calculates the square root of a number.
  • Useful in geometry, physics, and statistics.

7. Math.max() and Math.min():

  • Math.max(): Returns the largest of the given numbers.
  • Math.min(): Returns the smallest of the given numbers.
  • Finding highest/lowest values in a list.

8. Math.pow():

  • Calculates the power of a number (base raised to the exponent).
  • Useful in calculations involving Exponents, formulas, interest calculations.

9. Math.PI and Math.E:

  • Math.PI: Represents the mathematical constant π (approximately 3.14159).
  • Math.E: Represents Euler's number e (approximately 2.71828), the base of natural logarithms.
  • Used in formulas involving circles, trigonometry, and natural logarithms.

10. Math.sin(), Math.cos(), and Math.tan():

  • Math.sin(): Returns the sine of a number (in radians).
  • Math.cos(): Returns the cosine of a number (in radians).
  • Math.tan(): Returns the tangent of a number (in radians).
  • Used in trigonometric calculations, geometry, and physics.

11. Math.log(), Math.log10(), and Math.exp():

  • Math.log(): Returns the natural logarithm (base e) of a number.
  • Math.log10(): Returns the base-10 logarithm of a number.
  • Math.exp(): Returns e raised to the power of a given number.
  • Used in exponential growth/decay calculations, scientific computations.
